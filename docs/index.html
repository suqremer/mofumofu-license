アプリについて

「うちの子免許証」は、犬・猫・うさぎ・インコなど、あらゆるペットのオリジナル免許証を作れるiOSアプリです。写真を選んでコスチュームを着せるだけで、本物そっくりの免許証が完成します。

📸

免許証作成

写真を選んで情報を入力するだけ。背景切り抜きツールも搭載。

👔

47種のコスチューム

学ラン・警察官・サンタなど、着せ替えて楽しめます。

📡

NFC迷子対策

NFCタグにペット情報を書き込み。お散歩中の安心に。
